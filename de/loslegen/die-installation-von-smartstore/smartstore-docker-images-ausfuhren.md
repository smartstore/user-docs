# Smartstore Docker-Images ausführen

Mit Docker können Anwendungen schnell und unabhängig vom Betriebssystem und der Umgebung bereitgestellt werden. Zu diesem Zweck werden die Anwendungen in sogenannte Images gepackt. Diese Images können weitergegeben und auch kopiert werden. Im Gegensatz zu virtuellen Maschinen sind Docker-Images viel schlanker.

Auf einsatzbereite Smartstore Docker-Images können Sie über [https://github.com/orgs/smartstore/packages](https://github.com/orgs/smartstore/packages) zugreifen. Smartstore-Docker-Images können jedoch auch [https://smartstore.atlassian.net/wiki/spaces/SMNET50/pages/1956122004](https://smartstore.atlassian.net/wiki/spaces/SMNET50/pages/1956122004) erstellt und verwendet werden.
