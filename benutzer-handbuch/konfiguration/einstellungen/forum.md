| Signaturen aktiviert | Benutzer können eine Signatur festlegen. Die Signatur erscheint am Ende eines jeden Betrags im Forum. |
| Relative Zeitformatierung aktiviert | Relative Zeitangaben aktivieren (z. B. vor 1 Stunde, vor 2 Monaten). |
| Anzahl der Benutzerbeiträge zeigen | Zeigt die Gesamtzahl der bereits verfassten Beiträge eines Benutzers an. |
| Editor | Editor für das Forum (sollte nicht im Echtbetrieb geändert werden). |
| Gäste können Beiträge verfassen | Gäste können ohne eine Registrierung Beiträge erstellen. |
| Gäste können Themen eröffnen | Gäste können ohne eine Registrierung neue Themen erstellen. |
| Benutzer können Beiträge ändern | Benutzer können selbst erstellte Beiträge bearbeiten. |
| Benutzer können Beiträge löschen | Benutzer können selbst erstellte Beiträge löschen. |
| Benutzer können Abonnements verwalten | Benutzer können Forum-Abonnements verwalten. |
| Seitengröße der Themenübersicht | Legt die Seitengröße für Themen im Forum fest. |
| Seitengröße der Beitragsübersicht | Legt die Seitengröße für Beiträge auf der Themenseite fest, z. B. 10 Beiträge pro Seite. |
| Seitengröße der Suchergebnisse | Legt die Seitengröße für Suchergebnisse fest. |

| Signaturen aktiviert | Benutzer können eine Signatur festlegen. Die Signatur erscheint am Ende eines jeden Beitrags im Forum. |
| Private Nachrichten zulassen | Legt fest, ob private Nachrichten zugelassen sind. |
| Alarm für private Nachrichten | Legt fest, ob der Benutzer auf neue private Nachrichten hingewiesen werden soll. |
| Benachrichtigung bei privaten Nachrichten | Benutzer können bei neuen Nachrichten benachrichtigt werden. |
| Feeds aktiviert | Aktiviert RSS-Feeds für jedes Forum. |
| Anzahl der Feeds | Die Anzahl der Themen, die im Feed für das Forum aufgenommen werden soll. |
| Feeds über aktive Diskussionen aktiviert | Aktiviert einen RSS-Feed für die aktuellen Diskussionen. |
| Anzahl der Feeds für aktive Diskussionen | Legt die Anzahl der Themen für den aktiven Diskussions-Feed fest. |
