# Tags verwalten

Produkt-Tags können bearbeitet und gelöscht werden, indem Sie im Administrationsbereich zu **Katalog > Tags** navigieren. Produkt-Tags sind ideal geeignet, um Produkte unabhängig von Warengruppen zu gruppieren. Tags werden auch in den Suchergebnissen berücksichtigt.

## Anwendungsszenario

Stellen Sie sich vor, Sie betreiben einen Onlineshop für Motorradkleidung. Sie führen auch Artikel, die mit bestimmten Rennfahrern assoziiert werden und über unterschiedliche Produktkategorien wie Jacken, Helme oder Mützen verteilt sind. Um diese zusammenzustellen, wollen Sie vermutlich nicht extra eine neue Warengruppe erstellen, sondern arbeiten stattdessen lieber mit Tags. Tags, die Sie Produkten zuordnen, werden direkt unter den Produktdetails aufgelistet. Wenn Kunden an einem der verknüpften Tags interessiert sind, (in unserem Beispiel wäre das also der Name eines Rennfahrers), dann können sie einfach auf den Tag klicken und sehen eine Liste mit allen Produkten, die mit diesem Tag verknüpft wurden.

## Ein Tag erstellen

Um ein Produkt-Tag zu erstellen, schreiben Sie einfach das Tag, welches mit dem Produkt verknüpft werden soll innerhalb des Konfigurationsbereichs des Produkts im Reiter **Allgemein**. Wenn Sie mit der Eingabe beginnen, erhalten Sie eine Liste mit bereits in Ihrem Shop existierenden Produkt-Tags, welche die Buchstaben enthalten, die Sie gerade getippt haben.

![](./attachments/produkt-tags.PNG)

Die am häufigsten verwendeten Tags werden in einer Tag-Cloud auf der Startseite Ihres Shops angezeigt, von diesen werden die am häufigsten genutzten Tags größer und fetter dargestellt als weniger oft verwendete Tags. Die Anzeige der Tags auf der Startseite kann unter **Konfiguration > Einstellungen > Katalog-Einstellungen**, Reiter **Allgemein** deaktiviert werden. Die Anzahl der angezeigten Tags wird unter dem Reiter **Produktlisten** bestimmt.
