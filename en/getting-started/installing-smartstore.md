# Installing Smartstore

Before installation, please check if your hosting or server meets the [prerequisites](https://smartstore.atlassian.net/wiki/spaces/SMNET60/pages/2511044897/Technology+Prerequisites#Software-prerequisites) for Smartstore installation.  
Here's how to prepare your hosting or server for Smartstore installation:

- [Preparing Shared Hosting](./installing-smartstore/preparing-shared-hosting.md)
- [Preparing Windows server](./installing-smartstore/preparing-windows-server-vps-cloud-or-dedicated.md)
- [Preparing Linux server](./installing-smartstore/preparing-linux-vps-cloud-or-dedicated-server.md)
- [Run Smartstore as Docker-Container](./installing-smartstore/running-docker-images.md)
- [Install Smartstore](./installing-smartstore/starting-smartstore-installation.md)

> [!INFO]
> ### How to Build the Source Code
> If you've downloaded the source code version of Smartstore you need to build the solution before you can upload it onto your server. For more information about this topic please read [https://smartstore.atlassian.net/wiki/spaces/SMNET60/pages/2511051439](https://smartstore.atlassian.net/wiki/spaces/SMNET60/pages/2511051439).
