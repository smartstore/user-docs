# Technology & Prerequisites

## Technology

- State of the art architecture thanks to .NET 9, Entity Framework Core 6 and Domain Driven Design
- Easy to extend and extremely flexible thanks to modular design
- Highly scalable thanks to full page caching and web farm support
- A powerful theming engine lets you create themes & skins with minimum effort thanks to theme inheritance
- Point&Click theme configuration
- Powerful and lightning-fast media manager
- Powerful rule system for visual business rule creation
- Consistent and sophisticated use of modern components such as Vue.js, Sass, Bootstrap & more in the front and back end
- Easy shop management thanks to modern and clean UI

## Software prerequisites

- IIS 8+ (Integrated Pipeline mode)
- NET Core 9 (if selfcontained release: **Windows**: [.NET Core Hosting Bundle](https://learn.microsoft.com/en-us/aspnet/core/host-and-deploy/iis/hosting-bundle?view=aspnetcore-9.0) or **Linux**: nothing)
- **Windows Server** 2016 or higher *or*
- **Ubuntu** 16.04 or higher *or*
- **Debian** 8 or higher
- **Microsoft SQL Server** 2016 Express (or higher) for Windows *or*
- **Microsoft SQL Server 2019 for Linux** *or*
- **MySQL 8.0** or higher for Linux or Windows

## Hardware prerequisites on VPS, Cloud-Server or dedicated Server

- Minimum of 2 Cores
- Minimum of 2 GB RAM
- At least 100 MB database for an installation with demo data
- At least 500 MB of hard disk space for an installation with demo data

## Hardware prerequisites on shared Hosting

- Minimum of 1 Core
- Minimum of 1 GB RAM
- At least 100 MB database for an installation without demo data
- At least 500 MB of hard disk space for an installation with demo data
