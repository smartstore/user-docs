# Setting up Activity Types

In a shop, there are several activities that occur multiple times every day. Smartstore logs them in the **Activity Log**, which you can view by navigating to **Customers > Activity Log**.Â There are several activity types that are created during the installation of Smartstore. However, you can decide which activities are important to you and hence should be logged. You can turn activities on and off by navigating to **Configuration > Activity Types**. You can enable or disable them by checking or unchecking the value **Is Enabled** in the data grid and clicking **Save** afterwards.

## Some Interesting Activity Types

|     |     |
| --- | --- |
| Public store. Add product review | Adds an activity to the log every time a product review is added to one of your products. |
| Public store. Add forum post | Adds an activity to the log every time a forum post has been added to your forum. |
| Public store. Add to wishlist | Adds an activity to the log every time a product has been added to the wishlist by one of your customers. |
| Public store. Add blog comment | Adds an activity to the log every time a blog entry has been commented on. |
