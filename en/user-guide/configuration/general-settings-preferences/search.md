# Search

In the section **Search,** you can edit global settings relating to searching products.

|     |     |
| --- | --- |
| Search mode | Specifies the search mode. Please keep in mind that the search mode can - depending on catalog size - strongly affect search performance. **Is equal to** is the fastest, **Contains** the slowest. |
| Search fields | Specifies additional search fields. The product name is always searched. |
| Enable Instant Search | Activates Instant Search (Search-As-You-Type). Search hits and suggestions are already displayed before user finishes typing the search term. |
| Number of products | Specifies the number of product hits displayed in instant search. |
| Minimum search term length | Specifies the minimum length of a search term from which to show the result of instant search. |
| Result filtering | Activates or deactivates the filtering of search results by **Brand**, **Price**, **Rating** and **Delivery Time**. You can also specify the order of their appearance and a SEO friendly alias that is used for filter values in search URLs.<br><br>> [!INFO]<br>> Category filters are always displayed. You cannot deactivate them. |
| Minimum hit count for filters | Specifies the minimum number of search hits from which to show a filter. |
| Maximum number of filters | Specifies the maximum number of filters displayed per group. |
