# Blog Settings

![](./attachments/blog%20settings.jpg)

|     |     |
| --- | --- |
| Blog Enabled | Check the box to enable the blog in your store. |
| Posts page size | Set the page size for posts. |
| Allow Non-Registered Users to Leave Comments | Check the box to allow non-registered users to leave comments. |
| Notify About New Blog Comments | Check the box to notify the store owner about new blog comments. |
| Number of Tags (cloud) | The number of blog tags that appear in the tag cloud. |
| Display blog RSS feed link in the browser address bar | Check the box to enable the blog RSS feed link in the customer's browser address bar. |
