# News Settings

![](./attachments/news%20settings.jpg)

|     |     |
| --- | --- |
| News Enabled | Check the box if you want to enable news in your store. |
| Allow Non-Registered Users To Leave Comments | Check the box to allow non-registered users to leave comments. |
| Notify About New News Comments | Check the box to notify the store owner about new news comments. |
| Show On Home Page | Check the box to display your news items on your store home page. |
| Number Of Items To Display | The number of news items to display on your home page. |
| News Archive Page Size | The number of news items displayed per page when browsing the news archive. |
| Display News RSS Feed Link In The Browser Address Bar | Check the box to enable the news RSS feed link in the customer's browser address bar |
