# Reward Point Settings

![](./attachments/reward%20points1.jpg)

|     |     |
| --- | --- |
| Enabled | Check the box if you want to enable the reward points program. |
| Exchange Rate | Specify reward points exchange rate. |
| Points for Registration | Specify the number of points awarded for registration. |
| Points for a Product Review | Specify the number of points awarded for adding an approved product review. |
| Points for Purchases | Specify the number of points awarded for purchases. |
| Awarded Order Status | Points are awarded when the order status is… |
| Canceled Order Status | Points are canceled when the order status is… |
