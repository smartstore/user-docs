# Shopping Cart Settings

![](./attachments/Shoping%20Cart1.jpg)

|     |     |
| --- | --- |
| Display Cart After Adding Product | If the box is checked, customers will be taken to the shopping cart page immediately after adding a product to their cart. If unchecked, customers will remain on the same page where they are adding the product to the cart. |
| Maximum Shopping Cart Items | Maximum number of distinct products allowed in a shopping cart. |
| Show Product Images On Cart | Determines whether product images should be displayed in your store's shopping cart. |
| Show Product Images Of Bundle Items On Cart | Determines whether product images of bundle items should be displayed in your store's shopping cart. |
| Show Discount Box | Check the box if you want the discount coupon box to be displayed on the shopping cart page. |
| Show Gift Card Box | Check the box if you want the gift card coupon box to be displayed on the shopping cart page. |
| Number of 'Cross-Sells' | The number of ‘Cross-Sells’ to display on the shopping cart page. Enter 0 if you don’t want to load cross-sells. |
| Show Mini-Shopping Cart | Check the box to enable the mini-shopping cart. |
| Mini-Shopping Cart Product Number | Specifies the maximum number of products that can be displayed in the mini-shopping cart block. |
| Show Product Images In Mini-Shopping Cart | Determines whether product images are displayed in the mini-shopping cart block. |
| Show Legal Hints In Order Summary On The Confirm Order Page | Determines whether hints are shown in the order summary on the confirm order page. This text can be altered in the language resources. |
| Display Delivery Times | Determines whether delivery times are displayed in the shopping cart. |
| Display Product Short Description | Determines whether to display the product short description in the order summary. |
| Display Base Price | Determines whether the base price is displayed in the shopping cart. |
| Show Product Weight In Shopping Cart | Determines whether the product weight is shown in the shopping cart. |
| Show Quantity Of Linked Product | Determine whether the quantity of linked products appears at a variant attribute value. |
| Show Comment Box On Confirm Order Page | Determines whether the comment box is displayed on confirm order page. |
| Round Prices During Calculation | Determines whether the shop calculates with rounded price values. |

## Wishlist

![](./attachments/Shoping%20Cart2.jpg)

|     |     |
| --- | --- |
| Display Wishlist After Adding Product | If the box is checked, customers will be taken to the wishlist page immediately after adding a product to their wishlist. If unchecked, customers will remain on the page where they are adding the product to the wishlist. |
| Allow 'Out Of Stock' Items To Be Added To Wishlist | Check the box to allow ‘out of stock’ products to be added to the wishlist. |
| Maximum Wishlist Items | Maximum number of distinct products allowed in a wishlist. |
| Show Product Images On Wishlist | Determines whether product images are displayed on customer wishlists. |
| Show Product Images Of Bundle Items On Wishlist | Determines whether product images of bundle items are displayed on customer wishlists. |
| Allow Customers To Email Their Wishlists | Check the box to allow customers to e-mail their wishlists to friends. |
| Allow Guests To Email Their Wishlists | Check the box to allow guests to e-mail their wishlists to friends. |
