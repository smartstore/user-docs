# Things to know

### Keyboard shortcuts

CTRL+S: Save

ENTER: Confirm, open block editing mode

DEL: Delete

CTRL+Z: Undo

CTRL+Y: Repeat

CTRL+ALT+P: Preview mode

CTRL+F11: Full screen mode
