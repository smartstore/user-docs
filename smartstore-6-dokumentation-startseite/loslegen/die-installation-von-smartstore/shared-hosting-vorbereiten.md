# Shared Hosting vorbereiten

## Windows Shared Hosting

Smartstore läuft auf jedem Windows Shared Hosting, das [ASP.NET](http://ASP.NET) Core Runtime 9 unterstützt. Bitte erkundigen Sie sich bei Ihrem Hoster, ob dies der Fall ist.

Kopieren Sie einfach die Installationsdateien in das vom Hoster angegebene Webverzeichnis. In den meisten Fällen benötigen Sie dazu ein FTP-Programm. Befinden sich die Installationsdateien in einem ZIP-Archiv, muss der Inhalt entpackt und dann übertragen werden. Wenn die Dateien vollständig übertragen worden sind, können Sie mit der Installation fortfahren.

## Linux Shared Hosting

Um .NET Anwendungen wie Smartstore auf einem Shared-Linux-Hosting laufen zu lassen, sind "root"- (oder sudo)-Rechte für das Shared Hosting erforderlich. Da das Angebot sehr begrenzt ist und sich die zusätzlichen Kosten für einen Linux-Server in Grenzen halten, empfehlen wir die Nutzung eines Linux-Servers.
