# Eine Einführung in Smartstore

## Überblick

**Smartstore** ist eine kostenlose, quelloffene, voll funktionsfähige E-Commerce-Lösung für Unternehmen jeder Größe. Sie ist mit den Webstandards kompatibel und beinhaltet die neueste Microsoft Webtechnologie.

**Smartstore enthält alle wesentlichen Funktionen, um mehrsprachige und mehrwährungsfähige Shops für Desktop- oder Mobilgeräte zu erstellen** und ermöglicht SEO-optimierte Produktkataloge mit Unterstützung für eine unbegrenzte Anzahl von Produkten und Kategorien, Varianten, Paketen, Datenblättern, ESD, Rabatten, Gutscheinen und vielem mehr.

Ein umfassender Satz von Tools für CRM & CMS, Vertrieb, Marketing, Zahlung & Versand/Handling, etc. macht Smartstore zu einer leistungsstarken All-in-One-Lösung, die alle Ihre Bedürfnisse erfüllt... völlig kostenlos!

**Smartstore liefert ein wunderschönes und konfigurierbares Shop-Frontend, das mit einem Design-Ansatz auf höchstem Niveau entwickelt wurde,** einschließlich Komponenten wie `Bootstrap`, `SASS` und anderen. Die innovative ShopBar, eine leistungsstarke Layer-Navigation, Bewertungen, ein optimierter Checkout-Prozess... all das sorgt für ein perfektes Einkaufserlebnis für Ihre Kunden.

Die hochmoderne Architektur von Smartstore - mit `ASP.NET Core 9`, `Entity Framework Core 9` und Domain Driven Design Ansatz - macht es einfach zu erweitern, extrem flexibel und es macht im Grunde genommen Spaß, damit zu arbeiten.

## Funktionen

### Allgemeine Funktionen

|     |     |
| --- | --- |
| Kunden-Login/Mein Konto | Erlaubt jedem Besucher, der sich im Shop registriert, einen Kundenaccount zu eröffnen. Mit dem Konto können Kunden Adressen eingeben, Aufträge einsehen und den Versandstatus verfolgen. |
| SmartStore.biz Import | Erlaubt die einfache Übertragung nahezu aller Daten (Informationen zum genauen Umfang des Datenimports finden Sie [hier](http://documents.smartstore.com/SmartStoreNET_BIZ-Importer.pdf)) aus Ihrem vorhandenen **SmartStore.biz** Shop. |
| Lagerbestandverwaltung | Erlaubt Ihnen, einen Überblick darüber zu erhalten, welche Produkte neu bestellt werden müssen. |
| Auftags- und Kundenverwaltung | Erlaubt Ihnen, Rechnungen zu drucken und Produktbewertungen und Blogeinträge zu verwalten. |

### Kundenverwaltung

|     |     |
| --- | --- |
| Verwaltung von Kundengruppen | Erlaubt Ihnen, Kundengruppen zu definieren und unterschiedliche Preise für unterschiedliche Kundengruppen anzuzeigen. |
| Umfragen | Erkaubt die Erstellung von Umfragen. So können Sie Feedback zu Ihren Produkten und Werbeaktionen erhalten. |
| Geschenkgutscheine | Bietet Kunden die Möglichkeit, Geschenkgutscheine für Ihren Shop zu erwerben. |
| Wunschzettel | Bietet Ihren Kunden die Möglichkeit, eigene Wunschlisten zu erstellen und diese bequem an Freunde und Bekannte zu verschicken. |

### Produktverwaltung

|     |     |
| --- | --- |
| Spezifikationsattribute | Erlaubt Ihnen ausführliche Produktattribute für einen genauen Überblick zu erstellen. |
| Ausführliches Varianten-System | Ermöglicht dem Shop-Betreiber unterschiedliche Varianten von Produkten anzulegen. |
| Produktvergleich | Bietet Kunden die Möglichkeit Produkte in Ihrem Shop zu vergleichen. |
| Mehrere Shops | Erlaubt die Verwaltung von Produkten über mehrere Shops hinweg. |
| Elektronische Downloads (ESD) | Erlaubt eine konfigurierbare Download-Erlaubnis für virtuelle Produkte, einschließlich der optionalen Auslieferung der Lizenzdaten und Benutzungsvereinbarungen. |

### Mobile Geräte

|     |     |
| --- | --- |
| 100 % optimiert für mobile Geräte | Der Shop bemerkt automatisch, ob Ihre Kunden Ihren Shop über ein mobiles Gerät oder einen Rechner aufrufen und erstellt automatisch eine optimierte Darstellung. |
| SMS Benachrichtigung | Erlaubt Ihnen, Ihre Kunden darüber zu informieren, sobald ein vergriffenes Produkt wieder lieferbar ist. |

### Marketing

|     |     |
| --- | --- |
| E-Mail-Marketing | Erlaubt E-Mail-Marketing mit Newslettern mit frei konfigurierbaren Templates. |
| Bonuspunkte und Treueprogramm | Bietet Bonuspunkte und ein Treueprogramm zur Kundenbindung und als Anreiz für Ihre Kunden weitere Käufe zu tätigen. |
| Produkt Rezensionen und Bewertungen | Erlaubt Ihnen, Ihre Verkäufe zu erhöhen und ermutigt Ihre Kunden, Einkäufe mit Produktempfehlungen zu tätigen. |
| Cross-Promotionen | Ermöglicht intelligente Cross-Promotion für zuvor angesehene Produkte. |
| Exportiere zu Shopping Portalen | Bietet Schnittstellen für Preisvergleichportale: Google Base, billiger, ShopWahl, etc. |

### Versand

|     |     |
| --- | --- |
| Vorbestellungen | Mit Smartstore bieten Sie Ihren Kunden die Möglichkeit Produkte vorzubestellen, auch wenn diese noch nicht erhältlich sind. |
| Verwaltung von Retouren / RMA | Erlaubt die Verwaltung von zurückgesandten Waren. |
| Versandverfolgung | Kunden können den Versandstatus Ihrer Bestellungen online einsehen. |
| Eingabefelder während des Bestellvorgangs | Frei einstellbare Eingabefelder während des Bestellvorgangs (z. B. anklickbares Kästchen für Geschenkverpackungen). |

### SEO

|     |     |
| --- | --- |
| Meta Tags | Legen Sie konfigurierte Meta- und Title-Tages für jedes Produkt, jede Kategorie und jeden Hersteller an. |
| News auf der Startseite anzeigen | Stellen Sie sicher, dass die News automatisch auf Ihrer Startseite angezeigt werden, damit Suchmaschinen Ihre Inhalte als neu einstufen und Ihre Seite so einen höheren Rang beimessen. |
| Suchmaschinenfreundliche URLs | Dank sogenannter sprechender URLs erhalten Sie Top-Ratings von allen relevanten Suchmaschinen. Sie können suchmaschinenfreundliche Seitennamen für Inhalte wie Produkt- und Kategorieseiten einstellen. |

### Bezahl-Schnittstellen

|     |     |
| --- | --- |
| Bezahlung bei Veraand, Vorkasse, Kreditkarte | In Smartstore stehen Ihnen alle gängigen Bezahlmethoden wie Nachname, Vorkasse oder Kreditkarte zur Verfügung. Daneben können Sie auch die Online-Bezahlmethoden anbieten. |
| Amazon Bezahlungen | Sich mit Amazon einzuloggen und zu bezahlen, macht es für hunderte von Millionen von Amazon-Kunden einfach, in Ihrem Shop einzukaufen, da Ihre Kunden so schlicht über die bei Amazon gespeicherten Bankdaten ihre Einkäufe bezahlen können. |
| PayPal Express | PayPal Express bietet Ihren Kunden eine besonders schnelle Bezahlmethode, wodurch der Einkaufsvorgang verkürzt wird, was in der Folge zu weniger Abbrüchen beim Bestellvorgang führt. |
| Und vieles mehr | Für weitere Bezahlschnittstellen besuchen Sie bitte den [Smartstore Community Marketplace](http://community.smartstore.com/index.php?/files/). |
