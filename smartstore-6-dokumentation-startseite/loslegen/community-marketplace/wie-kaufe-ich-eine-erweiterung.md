# Wie kaufe ich eine Erweiterung?

Um ein eine Erweiterung (Plugin, Theme oder Sprachressource) für Smartstore zu kaufen, benötigen Sie einen PayPal-Account. Gehen Sie für den Kauf einfach in die Detailansicht der Erweiterung und drücken Sie auf den Button **In den Warenkorb**. Sie werden nun aufgefordert, Ihre Shop-URL anzugeben. Diese ist notwendig, da die Erweiterung nur für diese eine URL registriert wird Sie werden nun zum Warenkorb weitergeleitet, in dem Sie eine Übersicht aller ausgewählten Erweiterungen erhalten. 

Klicken Sie nun auf den Button **Zur Kasse**, akzeptieren Sie unsere AGB und klicken Sie auf **Kaufen**. Sie werden nun zu PayPal weitergeleitet, um den Bestellvorgang abzuschließen und die Zahlung durchzuführen. Sobald die Zahlung durchgeführt wurde, werden Sie in den [Smartstore Community Marketplace](http://community.smartstore.com/marketplace) zurückgeleitet und können sich die Erweiterung herunterladen.  

Eine Übersicht aller erfolgten Käufe finden Sie, wenn Sie oben rechts in der Community auf Ihr Profil klicken und den Eintrag **Kundenmenü** wählen. Im Reiter **Einkäufe** finden Sie eine Übersicht aller gekauften Erweiterungen. In der Detailansicht finden Sie neben anderen Informationen auch den Link, über den Sie das Plugin auf Ihren Rechner herunterladen können.
