# Shops verwalten

Sie können in Smartstore mehrere Shops anlegen und verwalten, indem Sie zu **Konfiguration > Shops** navigieren. Für weitere Informationen über die Arbeit in einer Multishop-Umgebung, lesen Sie bitte das Thema [Mit mehreren Shops arbeiten](../../benutzer-handbuch/allgemeine-konzepte/mit-mehreren-shops-arbeiten.md).

## Ansicht Shopdetails

|     |     |
| --- | --- |
| Shop-Name | Der Name Ihres Shops. Er wird an vielen Stellen genutzt, z. B. um Meta-Titel zu erstellen oder um als Überschrift für die physische Adresse Ihres Shops angezeigt zu werden. |
| Shop-URL | Die URL Ihres Shops, z. B. [http://www.yourstore.com/.](http://www.yourstore.com/) |
| HOST Werte | Eine durch Kommata getrennte Liste von möglichen HTTP\_POST Werten (z. B. "[yourstore.com](http://yourstore.com),[www.yourstore.com](http://www.yourstore.com)"). Diese Eigenschaft ist nur nötig, wenn Sie eine Multi-Store-Solution nutzen, um den vorliegenden Shop festzulegen. |
| Content Delivery Network URL | Die URL Ihres CDN, z. B. [https://xxx.cloudfront.net](https://xxx.cloudfront.net) oder [http://xxx.cloudflare.net](http://xxx.cloudflare.net). Diesen Wert festzulegen, erlaubt es Ihrer Seite statische Inhalte wie Mediafiles über das CDN auszuliefern. |
| ID des HTML Bodys | Erlaubt im Shop den Einsatz von angepassten CSS und Javascript. |
| Anzeigereihenfolge | Die Anzeigereihenfolge für den Shop. 1 steht für den Anfang der Liste. |
