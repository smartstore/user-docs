# Versand-Einstellungen

![](./attachments/versandeinstellungen.PNG)

|     |     |
| --- | --- |
| Kostenloser Versand ab Bestellwert aktivieren | Legt fest, ob der Versand einer Bestellung ab einem bestimmten Bestellwert kostenlos ist. |
| Schätzung der Versandkosten aktivieren | Benutzer können die Versandkosten schätzen lassen. Wenn diese Option ausgewählt, wird bei der Auftragszusammenfassung eine Option für den Kunden sichtbar, mit der er die Versandkosten schätzen lassen kann. Normalerweise können die Versandkosten nicht auf dieser Seite berechnet werden, da die Versandadresse noch nicht eingegeben wurde. Daher muss der Kunde vorher einige ortsabhängige Angaben machen, bevor die Versandkosten des Auftrags für ihn oder sie geschätzt werden können. |
| Versandereignisse anzeigen | Zeigt den Versand-Status auf der Versanddetails-Seite an, sofern vom Versandkosten-Modul unterstützt. |
| Versandartauswahl nur anzeigen, wenn mehr als eine Versandart zur Verfügung steht<br><br> [](https://wolfgang.smartstore.com/Admin/Setting/Shipping) | Legt fest, ob die Versandartauswahl nur im Checkout-Prozess angezeigt wird, wenn mehr als eine Versandart zur Verfügung steht |
| Nur den höchsten Transportzuschlag berechnen<br><br> [](https://wolfgang.smartstore.com/Admin/Setting/Shipping) | Bestimmt ob bei der Berechnung der Versandkosten nur der höchste Transportzuschlag von Produkten berücksichtigt wird. |
| Land | Land eingeben. |
| Bundesland / Region | Bundesland oder Region eingeben. |
| PLZ | Postleitzahl eingeben. |
