# Bonuspunkte-Einstellungen

![](./attachments/bonuspunkte-einstellungen.PNG)

|     |     |
| --- | --- |
| Aktiviert | Legt fest, ob Bonuspunkte aktiviert werden. |
| Wechselkurs | Legt den Wechselkurs für Bonuspunkte fest. |
| Punkte abrunden | Legt fest, ob bei der Punkteberechnung abgerundet werden soll. Ansonsten werden Bonuspunkte aufgerundet. |
| Punkte für die Registrierung | Bonuspunkte, die für eine Registrierung gewährt werden. |
| Punkte für eine Produkt-Rezension | Bonuspunkte, die für das Verfassen einer genehmigten Produkt Rezension gewährt werden. |
| Punkte auf den Bestellwert | Bonuspunkte, die für den Einkauf gewährt werden. |
| Voraussetzung Auftragsstatus | Die Bonuspunkte werden gewährt, wenn der Auftragsstatus ist ... |
| Stornierter Auftragsstatus | Die Bonuspunkte werden storniert, wenn der Auftragsstatus ist ... |
