# Auftrags-Einstellungen

![](./attachments/auftragseinstellungen.PNG)

|     |     |
| --- | --- |
| "Erneut bestellen" aktiviert | Benutzer können Bestellung erneut abschicken. |
| Mindestbestellwert | Legt den standardmäßigen Mindestbestellwert für den Shop fest. Wird von Beschränkungen in Kundengruppen überschrieben. |
| Höchstbestellwert | Legt den standardmäßigen Höchstbestellwert für den Shop fest. Wird von Beschränkungen in Kundengruppen überschrieben. |
| Kundengruppen erweitern den Wertebereich | Legt fest, ob mehrfache Bestellwertbeschränkungen durch Kundengruppenzuordnungen den erlaubten Bestellwertbereich erweitern. |
| Gäste dürfen bestellen | Gästen erlauben, ohne sich zu registrieren, Bestellungen aufzugeben. |
| Allg. Geschäftsbedingungen aktiviert | Benutzer müssen die allg. Geschäftsbedingungen akzeptieren, bevor sie einen Auftrag erteilen können. |
| "Auftrag abgeschlossen" Seite unterbinden | Der Kunde wird direkt auf die Auftrags-Detail-Seite geleitet, falls diese Einstellung aktiviert ist. |
| Geschenkgutschein ist aktiviert, wenn Auftragsstatus.... | Geschenkgutscheine sind deaktiviert, wenn der Auftragsstatus ist … |
| Geschenkgutschein wird deaktiviert, wenn Auftragsstatus... | Geschenkgutscheine sind deaktiviert, wenn der Auftragsstatus ist ... |
| Anzahl der Aufträge pro Seite<br><br> [](https://wolfgang.smartstore.com/Admin/Setting/Order) | Legt die Anzahl der dargestellten Aufträge pro Seite fest. |
| Bestellnummer | Legen Sie einen Startwert für Bestellnummern fest. Dies ist nützlich, wenn Sie ihre Aufträge ab einem bestimmten Wert starten lassen möchten. Der Wert muss größer sein, als die aktuell höchste Bestellnummer und gilt für zukünftige Bestellungen. |
| Aufträge aller Shops anzeigen | Legt fest, ob dem Kunden die Aufträge aller Shops angezeigt werden sollen. Ist diese Option deaktiviert, so werden nur die Aufträge des aktuellen Shops angezeigt. |

## Retoureneinstellungen

![](./attachments/retoureneinstellungen.PNG)

|     |     |
| --- | --- |
| Retourenanträge aktiviert | Benutzer können Rücksendewünsche erteilen. |
| Erlaubter Zeitraum für Retourenwünsche in Tagen | Anzahl der Tage, an denen ein Link für einen Rücksendewunsch im Kundenbereich sichtbar ist. |
| Retourengründe | Eine kommaseparierte Liste von möglichen Retourengründen, die der Benutzer auswählen kann, wenn er einen Rücksendewunsch übermittelt. |
| Gewünschte Aktion für Retoure | Eine kommaseparierte Liste von Aktionen, die der Benutzer auswählen kann, wenn er einen Rücksendewunsch übermittelt, z. B. "Ersatz", "Gutschein" u.s.w. |
