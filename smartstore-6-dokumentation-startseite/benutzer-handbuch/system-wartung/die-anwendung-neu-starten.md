# Die Anwendung neu starten

Manchmal müssen Sie die Anwendung Ihres Shops neu starten, z. B., wenn Ihr Shop auf die aktuellste Version von Smartstore aktualisiert wurde. Sie können die Anwendung neu starten, indem Sie auf das Zahnrad-Symbol in der oberen rechten Ecke der Administrations-Ebene klicken und **Neustart** anklicken.

![](./attachments/cache-loeschen.PNG)

## Gelegenheiten, bei denen Sie die Anwendung neustarten müssen

- Veränderte Einstellung für eine Aufgabe
- Nach dem Update auf die neue Version
